# flower-data-augmentation
# ​ Flower Dataset - Data Augmentation

## Introduction
Augmenting a dark red flower image dataset to increase diversity and improve model performance.

## Dataset
- Source folder: `dataset/original/`
- Contains images of dark red flowers.

## Augmentation Techniques
1. Rotation: ±20°
2. Width/Height shift: up to 10%
3. Shear: up to 20°
4. Zoom: ±20%
5. Horizontal flip: 50% probability
6. Brightness adjustment: 0.7×–1.3×
